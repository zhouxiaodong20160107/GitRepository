1. 安装完成后向服务器注册本机配置
注册本机账户： git config --global user.name "zxd"
注册本机email： git config --global user.email "z736262989@qq.com"

2.创建一个GIT版本库
1)创建：mkdir gitRepository
2) 进入：cd gitRepository
3) 显示地址： pwd  

4.清空仓库
1）git init

5. 添加文件到GIT版本中
1)先把文件放到仓库文件夹中,例如 git1.txt；
2）git add git1.ext
3)  告诉仓库 需要提交文件：commit -m "commit the first git file"
提示：一次可以用ADD添加多个文件，可以一次性执行 commit 命令，都提交；

5.版本管理系统只能跟踪文本文件的改的，二进制的变化记录不了，二进制变化如视频音频
